1. Visao Geral

O Musion e um aplicativo mobile para usuarios que gostam de avaliar albuns, escrever reviews e descobrir musica por meio da comunidade. Ele cria uma camada social sobre o consumo musical, reunindo feed, perfil, comentarios, curtidas, chat, notificacoes, compartilhamento e recomendacoes personalizadas.

O problema resolvido e a falta de um espaco focado em reviews musicais dentro das plataformas tradicionais de streaming. O Musion organiza opinioes por usuario, album e comunidade.

3. Arquitetura

A solucao e dividida entre app mobile e backend. O mobile foi desenvolvido em React Native com Expo. A API foi desenvolvida em NestJS, usando Prisma como ORM e MySQL como banco SQL.

Camada Tecnologias Responsabilidade
Mobile React Native, Expo, React Navigation, Axios, AsyncStorage Interface, navegacao, consumo da API, experiencia mobile e sensor.
Backend NestJS, Prisma, JWT, bcrypt, Nodemailer Autenticacao, regras de negocio, dados, moderacao e integracoes.
Banco MySQL Persistencia de usuarios, reviews, comentarios, mensagens e denuncias.
Integracoes Spotify, Last.fm, Deezer/fallback, Cloudinary, SMTP, Google Catalogo musical, recomendacoes, previews, imagens e login.

5. Big Data, IA e Recomendacoes

O Musion utiliza uma abordagem de recomendacao hibrida. A API analisa reviews recentes, notas altas, albuns ja avaliados, atividade da comunidade e dados externos de Spotify e Last.fm. O objetivo e gerar sugestoes mais coerentes e reduzir repeticao.

Essa camada funciona como uma API inteligente de insights musicais, consumida pela aba Trending do aplicativo.

6. Sensor Mobile

Na tela de adicionar review, o usuario pode ativar o controle por sensor. A inclinacao do celular, capturada pelo acelerometro, move o slider de nota. Isso demonstra uso de dados de hardware em tempo real dentro da aplicacao mobile.
